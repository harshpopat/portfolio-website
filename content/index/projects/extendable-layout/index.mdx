IT Logger is used by technical people or engineers who provide customer support or IT support in a company. This web app is used to log the issues of the users and to whom it is assigned. It also allows to label those as solved or unsolved. Using this the engineers can keep track of all the issues which need to be addressed as well as those which are resolved.
